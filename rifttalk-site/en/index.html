How RiftTalk works

No magic — just careful automation around Discord and match status.

1

Link Discord

Authorization uses official Discord OAuth. RiftTalk never asks for your Discord password.

2

Match start

When the game starts, temporary voice channels are created and roles/permissions are granted to your team.

3

Auto cleanup

After the match ends, users are removed from channels, roles are revoked, and temporary channels are deleted.

FAQ

Answers to common questions.

Is it safe? Does RiftTalk ask for my Discord password?
Yes. Linking uses official Discord OAuth. RiftTalk never asks for or stores your Discord password.
Why am I not being moved to the voice channel?
Check: the app is running before the match starts, Discord is running and logged in, your account is linked, and you are on the correct Discord server.
When are channels created and removed?
Channels are created when the game starts. After the match ends, users are removed, roles are revoked, and temporary channels are deleted.
Do I need to install anything else?
No. Just run RiftTalk and have Discord and League of Legends installed.
Windows shows a security warning. Is this normal?
Yes. Windows/SmartScreen may warn about apps distributed outside the Microsoft Store. Download RiftTalk only from the official GitHub links and run it only if you trust the source.
Does RiftTalk read or store my League of Legends account data?
No. RiftTalk never asks for your LoL login/password and does not store LoL account data. It only uses local match state (League Client API/LCU) to detect when a game starts/ends.
